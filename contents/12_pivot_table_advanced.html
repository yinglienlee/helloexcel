📊 樞紐分析表進階應用

活用計算欄位、切片器、時程表與 Power Pivot,打造專業級動態儀表板

1計算欄位 (Calculated Field)

計算欄位是在樞紐分析表中建立自訂計算的強大工具,可以在不修改原始資料的情況下,直接在樞紐分析表內進行複雜運算。

💡 什麼是計算欄位?
計算欄位 (Calculated Field) 是根據樞紐分析表中已有的欄位,透過公式運算產生新的數值欄位。例如:利潤率 = 利潤 ÷ 營收、成長率 = (本期 - 上期) ÷ 上期 等。

建立計算欄位的步驟

  1. 點選樞紐分析表內的任一儲存格
  2. 前往「樞紐分析表分析」分頁 → 點選「欄位、項目與集」 → 選擇「計算欄位」
  3. 在「插入計算欄位」對話框中:
    • 名稱: 輸入新欄位的名稱 (例如:「利潤率」)
    • 公式: 輸入計算公式 (例如:=利潤/營收)
  4. 點選「新增」後,再點選「確定」
📌 操作範例:建立「利潤率」計算欄位
假設樞紐分析表中已有「營收」與「利潤」兩個數值欄位:

步驟 1: 點選樞紐分析表 → 樞紐分析表分析 → 欄位、項目與集 → 計算欄位
步驟 2: 名稱輸入「利潤率」
步驟 3: 公式輸入 =利潤/營收
步驟 4: 點選「新增」→「確定」

新增完成後,樞紐分析表會自動新增「利潤率」欄位,並顯示計算結果。

計算欄位的常用公式範例

計算欄位名稱 公式 說明
利潤率 =利潤/營收 計算每筆交易的利潤佔營收的百分比
平均單價 =銷售額/銷售數量 計算平均每件商品的售價
毛利 =營收-成本 計算營收扣除成本後的毛利潤
折扣金額 =原價*折扣率 計算實際的折扣金額
淨利 =營收-成本-費用 計算扣除所有成本與費用後的淨利潤
⚠️ 計算欄位的限制:
• 計算欄位只能參考數值欄位,無法參考列或欄標籤
• 公式會套用到樞紐分析表的所有資料列
• 若需要更複雜的邏輯判斷,建議先在原始資料中新增輔助欄位

2計算項目 (Calculated Item)

計算項目與計算欄位不同,它是在特定維度(列或欄標籤)中新增自訂項目,用於分組或建立小計。

💡 什麼是計算項目?
計算項目 (Calculated Item) 是在樞紐分析表的列或欄標籤中,根據現有項目建立新的分組項目。例如:將「第一季」定義為「1月+2月+3月」,或將「北部地區」定義為「台北+新北+桃園」。

建立計算項目的步驟

  1. 在樞紐分析表中,點選要新增計算項目的欄位標籤 (例如「月份」欄位中的任一月份)
  2. 前往「樞紐分析表分析」分頁 → 點選「欄位、項目與集」 → 選擇「計算項目」
  3. 在「插入計算項目」對話框中:
    • 名稱: 輸入新項目名稱 (例如:「第一季」)
    • 公式: 輸入項目公式 (例如:='1月'+'2月'+'3月')
  4. 點選「新增」後,再點選「確定」
📌 操作範例:建立「第一季」計算項目
假設樞紐分析表的列標籤為「月份」,包含 1月、2月、3月等項目:

步驟 1: 點選月份欄位中的任一項目 (例如「1月」)
步驟 2: 樞紐分析表分析 → 欄位、項目與集 → 計算項目
步驟 3: 名稱輸入「第一季」
步驟 4: 公式輸入 ='1月'+'2月'+'3月'
步驟 5: 點選「新增」→「確定」

新增完成後,樞紐分析表的月份列會出現新的「第一季」項目。

計算項目的實用範例

應用場景 計算項目名稱 公式範例
季度分組 第一季 ='1月'+'2月'+'3月'
區域分組 北部地區 ='台北'+'新北'+'桃園'
產品分類 高階產品 ='產品A'+'產品B'
年齡分組 青年客群 ='18-25歲'+'26-35歲'
⚠️ 計算項目 vs 計算欄位的差異:
• 計算欄位: 建立新的數值欄位,基於現有數值欄位運算
• 計算項目: 在特定維度中建立新的分組項目,基於現有項目加總或組合
• 計算欄位與計算項目不能同時存在於同一個樞紐分析表中

3切片器 (Slicer)

切片器是一種視覺化的互動式篩選工具,讓使用者可以透過點選按鈕快速篩選樞紐分析表的資料,無需使用下拉式選單。

💡 什麼是切片器?
切片器 (Slicer) 是一種圖形化的篩選按鈕面板,可以讓使用者快速篩選樞紐分析表或圖表的資料。相較於傳統的篩選器下拉選單,切片器更直覺、更易於操作,特別適合用於儀表板設計。

插入切片器的步驟

  1. 點選樞紐分析表內的任一儲存格
  2. 前往「樞紐分析表分析」分頁 → 點選「插入切片器」
  3. 在「插入切片器」對話框中,勾選要建立切片器的欄位 (例如:「月份」、「產品類別」、「業務員」等)
  4. 點選「確定」
  5. 切片器會以獨立的浮動視窗出現在工作表上,可自由拖曳調整位置與大小

切片器的操作與設定

單選 點選切片器中的任一項目,樞紐分析表會立即篩選顯示該項目的資料
多選 按住 Ctrl 鍵,可同時選取多個項目。或點選切片器右上角的「多重選取」按鈕 (圖示為三條橫線加勾選框)
清除篩選 點選切片器右上角的「清除篩選」按鈕 (圖示為漏斗加叉叉),恢復顯示所有資料
調整樣式 選取切片器後,前往「切片器」分頁 → 選擇不同的「切片器樣式」,可改變顏色與外觀
調整欄數 選取切片器 → 切片器分頁 → 「按鈕」群組 → 調整「欄」數量,可將切片器按鈕排列成多欄顯示
排序項目 點選切片器右上角的「⋮」(更多選項) → 選擇「遞增排序」或「遞減排序」
📌 實務應用:建立業務員切片器
假設樞紐分析表包含「業務員」、「月份」、「銷售額」等欄位:

步驟 1: 點選樞紐分析表 → 樞紐分析表分析 → 插入切片器
步驟 2: 勾選「業務員」欄位 → 確定
步驟 3: 切片器出現後,調整位置到樞紐分析表旁邊
步驟 4: 點選切片器中的任一業務員名稱,樞紐分析表立即顯示該業務員的銷售資料
步驟 5: 按住 Ctrl 可同時選取多位業務員進行比較

這樣就能快速切換檢視不同業務員的銷售績效!

切片器的進階功能

  • 連結多個樞紐分析表: 一個切片器可以同時控制多個樞紐分析表。選取切片器 → 切片器分頁 → 「報表連線」→ 勾選要連結的樞紐分析表
  • 搭配圖表使用: 切片器不僅適用於樞紐分析表,也可以搭配樞紐分析圖使用,提供互動式的資料視覺化體驗
  • 儀表板設計: 在儀表板中使用多個切片器 (例如:「月份」、「產品」、「地區」),讓使用者可以自由組合篩選條件,進行多維度分析

4時程表 (Timeline)

時程表是專為日期欄位設計的切片器,提供更直覺的時間篩選方式,可以快速選取特定日期範圍、月份、季度或年度。

💡 什麼是時程表?
時程表 (Timeline) 是針對日期欄位的專用篩選器,以時間軸的形式呈現,讓使用者可以透過拖曳滑桿或點選時間區間,快速篩選特定時間範圍的資料。時程表支援「日」、「月」、「季」、「年」四種時間單位的切換。

插入時程表的步驟

  1. 點選樞紐分析表內的任一儲存格
  2. 前往「樞紐分析表分析」分頁 → 點選「插入時程表」
  3. 在「插入時程表」對話框中,選擇要建立時程表的日期欄位 (例如:「訂單日期」、「交易日期」)
  4. 點選「確定」
  5. 時程表會以獨立視窗出現,可拖曳調整位置與大小

時程表的操作方式

切換時間單位 點選時程表右上角的下拉選單,可選擇「年」、「季」、「月」、「日」等不同時間單位顯示
選取單一時間 點選時程表中的任一時間區塊 (例如:「2024年3月」),樞紐分析表會篩選該時間的資料
選取連續範圍 點選起始時間後,按住滑鼠左鍵向右拖曳,可選取連續的時間範圍 (例如:「2024年1月~3月」)
清除篩選 點選時程表右上角的「清除篩選」按鈕,恢復顯示所有時間的資料
調整時間軸範圍 拖曳時程表底部的滑桿,可快速移動時間軸檢視不同時間區段
📌 實務應用:建立月份時程表分析營收
假設樞紐分析表包含「訂單日期」、「產品」、「營收」等欄位:

步驟 1: 點選樞紐分析表 → 樞紐分析表分析 → 插入時程表
步驟 2: 選擇「訂單日期」欄位 → 確定
步驟 3: 時程表出現後,點選右上角下拉選單,選擇「月」單位
步驟 4: 點選「2024年1月」,拖曳至「2024年3月」,選取第一季的資料
步驟 5: 樞紐分析表立即顯示第一季的營收統計

透過時程表,可以輕鬆切換不同時間區間,快速比較各期營收表現!

時程表的優勢與應用場景

  • 視覺化時間篩選: 相較於傳統的日期篩選器,時程表以時間軸方式呈現,更直覺易懂
  • 快速切換時間單位: 可在日、月、季、年之間自由切換,滿足不同層級的時間分析需求
  • 適合趨勢分析: 特別適用於銷售趨勢、財務報表、專案進度等需要時間序列分析的場景
  • 搭配切片器使用: 時程表可與其他維度的切片器 (如產品、地區) 搭配使用,建立多維度的動態儀表板
⚠️ 時程表的使用前提:
• 樞紐分析表的資料來源必須包含日期格式的欄位
• 若欄位為文字格式的日期 (如「2024/01/15」字串),需先轉換為日期格式才能使用時程表
• 時程表僅適用於單一日期欄位,若需篩選多個日期欄位,需建立多個時程表

5Power Pivot 介紹

Power Pivot 是 Excel 的強大增益集,能夠處理百萬列以上的大量資料,建立複雜的資料模型,並使用 DAX 公式進行進階計算。

💡 什麼是 Power Pivot?
Power Pivot 是 Excel 的進階商業智慧工具,主要功能包括:
• 處理大量資料: 可載入與分析數百萬列的資料,突破一般工作表的 1,048,576 列限制
• 建立資料模型: 可在 Excel 中建立多表關聯,類似資料庫的關聯式結構
• DAX 公式: 使用 Data Analysis Expressions (DAX) 語言建立複雜的計算欄位與量值
• 記憶體內運算: 資料壓縮儲存在記憶體中,查詢與計算速度極快

啟用 Power Pivot

📌 如何啟用 Power Pivot (適用 Excel 2016 及以上版本):

步驟 1: 點選「檔案」 → 「選項」
步驟 2: 選擇「增益集」
步驟 3: 在下方「管理」下拉選單選擇「COM 增益集」 → 點選「執行」
步驟 4: 勾選「Microsoft Power Pivot for Excel」 → 點選「確定」
步驟 5: Excel 功能區會出現新的「Power Pivot」分頁

註: Power Pivot 僅適用於 Office 專業增強版、Office 365 專業增強版等特定版本,家用版可能無此功能。

Power Pivot 的核心功能

1. 匯入多來源資料

Power Pivot 可以從多種來源匯入資料:

  • Excel 工作表或表格
  • SQL Server、Access 等資料庫
  • 文字檔 (CSV、TXT)
  • Azure、SharePoint 等雲端資料源
  • 網路資料摘要 (Data Feeds)

  • Excel 工作表或表格
  • SQL Server、Access 等資料庫
  • 文字檔 (CSV、TXT)
  • Azure、SharePoint 等雲端資料源
  • 網路資料摘要 (Data Feeds)
📌 建立資料表關聯的步驟:

步驟 1: 前往 Power Pivot 分頁 → 點選「管理」,開啟 Power Pivot 視窗
步驟 2: 點選「圖表檢視」按鈕,切換到圖表模式
步驟 3: 將兩個表格中相同意義的欄位 (如「產品代碼」) 用滑鼠拖曳連接

步驟 5: 關聯建立後,樞紐分析表就能跨表格進行分析

3. DAX 公式簡介

DAX (Data Analysis Expressions) 是 Power Pivot 的專用公式語言,語法類似 Excel 公式,但功能更強大。

DAX 函數類型 範例函數 說明
彙總函數 SUM, AVERAGE, COUNT 計算總和、平均、計數等基本統計
篩選函數 CALCULATE, FILTER 根據條件篩選資料後進行計算
時間智慧函數 TOTALYTD, SAMEPERIODLASTYEAR 計算年初至今、同比、環比等時間相關分析
關聯函數 RELATED, RELATEDTABLE 在關聯表格間擷取資料
DAX 範例 1:計算總營收 總營收 = SUM('銷售明細'[金額])DAX 範例 2:計算去年同期營收 去年營收 = CALCULATE(SUM('銷售明細'[金額]), SAMEPERIODLASTYEAR('日期'[日期]))DAX 範例 3:計算年成長率 年成長率 = DIVIDE([總營收] - [去年營收], [去年營收])

Power Pivot vs 傳統樞紐分析表

資料量限制 • 傳統樞紐: 受限於工作表的 1,048,576 列
• Power Pivot: 可處理數百萬至數千萬列資料
多表關聯 • 傳統樞紐: 僅能分析單一表格或範圍
• Power Pivot: 可建立多表關聯,進行跨表分析
計算能力 • 傳統樞紐: 僅能使用計算欄位/項目,功能有限
• Power Pivot: 支援 DAX 公式,可進行複雜的商業邏輯計算
效能 • 傳統樞紐: 資料量大時可能變慢
• Power Pivot: 記憶體內運算,大量資料仍能快速回應
適用場景 • 傳統樞紐: 適合中小型資料的快速分析
• Power Pivot: 適合企業級大數據分析、複雜的商業智慧應用

6實務範例:動態營收分析儀表板

以下將示範如何結合樞紐分析表、切片器與時程表,建立一個專業的動態營收分析儀表板,讓使用者可以依月份或業務員快速切換視圖。

範例資料結構

假設我們有一份「銷售記錄表」,包含以下欄位:

訂單編號 訂單日期 業務員 產品類別 產品名稱 銷售數量 單價 銷售額 成本
O-0001 2024/01/15 王小明 電子產品 筆記型電腦 2 25,000 50,000 35,000
O-0002 2024/01/18 李美玲 家電 咖啡機 5 8,000 40,000 28,000
O-0003 2024/02/02 王小明 電子產品 平板電腦 3 15,000 45,000 30,000

儀表板建立步驟

步驟 1:建立樞紐分析表

  1. 選取銷售記錄表的任一儲存格 → 插入 → 樞紐分析表
  2. 選擇「新工作表」,建立樞紐分析表
  3. 拖曳欄位設定:
    • 列: 業務員
    • 欄: (暫時留空)
    • 值: 銷售額 (加總)、成本 (加總)

步驟 2:新增計算欄位 - 利潤與利潤率

  1. 點選樞紐分析表 → 樞紐分析表分析 → 欄位、項目與集 → 計算欄位
  2. 新增「利潤」欄位:
    • 名稱: 利潤
    • 公式: =銷售額-成本
  3. 新增「利潤率」欄位:
    • 名稱: 利潤率
    • 公式: =利潤/銷售額
    • 設定數值格式為「百分比」,保留 1 位小數

步驟 3:插入切片器 - 業務員與產品類別

  1. 點選樞紐分析表 → 樞紐分析表分析 → 插入切片器
  2. 勾選「業務員」與「產品類別」→ 確定
  3. 調整切片器位置,放置在樞紐分析表上方或左側
  4. 調整切片器樣式:選取切片器 → 切片器分頁 → 選擇喜歡的配色
  5. 調整切片器欄數:切片器分頁 → 按鈕 → 欄 = 3 (讓業務員名稱並排顯示)

步驟 4:插入時程表 - 訂單日期

  1. 點選樞紐分析表 → 樞紐分析表分析 → 插入時程表
  2. 選擇「訂單日期」欄位 → 確定
  3. 調整時程表位置,放置在切片器旁邊
  4. 點選時程表右上角下拉選單,選擇「月」作為顯示單位

步驟 5:新增樞紐分析圖表

  1. 點選樞紐分析表 → 樞紐分析表分析 → 樞紐分析圖
  2. 選擇「直條圖」→ 確定
  3. 圖表會自動與樞紐分析表連動,切片器與時程表的篩選也會同步套用到圖表
  4. 調整圖表標題為「業務員營收分析」

步驟 6:美化儀表板版面

  1. 調整工作表名稱為「營收儀表板」
  2. 統一切片器與時程表的樣式配色
  3. 在儀表板上方新增標題:使用合併儲存格,輸入「📊 營收分析儀表板」,設定字型大小 18、粗體、置中
  4. 調整樞紐分析表的數值格式:銷售額、成本、利潤使用「會計」格式,千位分隔符號
  5. 隱藏格線:檢視分頁 → 取消勾選「格線」

儀表板的使用方式

  • 依業務員篩選: 點選切片器中的業務員名稱 (例如「王小明」),樞紐分析表與圖表立即顯示該業務員的銷售資料
  • 依產品類別篩選: 點選「電子產品」,查看電子產品的銷售表現
  • 依月份篩選: 使用時程表拖曳選取「2024年1月~3月」,分析第一季的營收
  • 組合篩選: 同時選取「王小明」+ 「電子產品」+ 「2024年1月」,進行多維度交叉分析
  • 清除篩選: 點選切片器或時程表的「清除篩選」按鈕,恢復顯示全部資料
💡 儀表板設計的最佳實務:
• 保持簡潔: 不要在單一頁面放置過多圖表,避免資訊過載
• 一致性: 統一配色、字型與樣式,讓視覺呈現更專業
• 互動性: 善用切片器與時程表,讓使用者能自主探索資料
• 關鍵指標突出: 使用粗體、顏色或儲存格格式突出重要數值 (如總營收、利潤率)
• 加入說明: 在儀表板上方或側邊加入簡短的使用說明,提升易用性

任務描述:

訂單日期 地區 產品類別 銷售額 成本

  7. 新增樞紐分析圖 (橫條圖),顯示各產品類別的利潤排名
  8. 設定儀表板標題「產品類別利潤分析儀表板」,統一樣式配色

進階挑戰:

  • 使用條件式格式,將利潤率 > 30% 的產品類別以綠色底色標示
  • 將切片器設定為「報表連線」,同時控制樞紐分析表與圖表
  • 在圖表上方新增文字方塊,顯示「總利潤」與「平均利潤率」(可使用 GETPIVOTDATA 函數擷取)

  • 使用時程表切換不同月份,觀察各組的銷售趨勢變化

練習題 3:Power Pivot 多表關聯分析 (進階)

任務描述:

某電商公司有三個資料表:「訂單明細表」、「產品主檔表」、「客戶主檔表」,請使用 Power Pivot 建立資料模型,進行跨表分析。

資料表結構:

訂單明細表 (50,000+ 筆)

訂單編號 訂單日期 客戶編號 產品代碼 數量 單價
O-100012024/01/05C001P-A0131,200
O-100022024/01/05C002P-B0518,500

產品主檔2:23 PM表

產品代碼 產品名稱 產品類別 成本
P-A01無線滑鼠電腦周邊800
P-B05咖啡機家電6,000

客戶主檔表

客戶編號 客戶名稱 地區 會員等級
C001台北科技公司台北金卡
C002高雄貿易行高雄銀卡

Power Pivot 建立步驟:

  1. 啟用 Power Pivot 增益集 (檔案 → 選項 → 增益集 → COM 增益集 → 勾選 Power Pivot)
  2. 將三個表格轉換為「表格」格式 (插入 → 表格),並命名為「訂單明細」、「產品主檔」、「客戶主檔」
  3. 前往 Power Pivot 分頁 → 點選「新增至資料模型」(依序將三個表格加入)
  4. 開啟 Power Pivot 視窗 → 點選「圖表檢視」
  5. 建立關聯:
    • 訂單明細[產品代碼] → 產品主檔[產品代碼]
    • 訂單明細[客戶編號] → 客戶主檔[客戶編號]
  6. 新增 DAX 計算欄位「銷售額」(在訂單明細表):
    銷售額 = '訂單明細'[數量] * '訂單明細'[單價]
  7. 新增 DAX 量值「總利潤」:
    總利潤 = SUM('訂單明細'[銷售額]) - SUM(RELATED('產品主檔'[成本]) * '訂單明細'[數量])

樞紐分析要求:

  1. 建立樞紐分析表,使用 Power Pivot 資料模型
  2. 設定欄位:
    • 列: 產品主檔[產品類別]、客戶主檔[地區]
    • 值: 銷售額、總利潤
  3. 插入「產品類別」切片器與「會員等級」切片器
  4. 插入「訂單日期」時程表,以「季」為單位
  5. 新增樞紐分析圖,顯示各產品類別在不同地區的銷售額分布

分析目標:

  • 找出銷售額最高的產品類別與地區組合
  • 比較金卡與銀卡會員的消費金額差異
  • 分析各季度的銷售趨勢,找出淡旺季
⚠️ 注意事項:
• 確認 Excel 版本支援 Power Pivot (Office 專業增強版或 Microsoft 365)
• 建立關聯前,確保關聯欄位的資料格式一致 (例如:產品代碼都是文字格式)
• DAX 公式區分大小寫,表格名稱與欄位名稱需正確輸入
• Power Pivot 處理大量資料時需要較多記憶體,建議電腦 RAM ≥ 8GB

🎓 學習重點回顧

  • 計算欄位: 在樞紐分析表中建立自訂公式,無需修改原始資料
  • 計算項目: 在特定維度中建立分組項目,例如季度、地區群組
  • 切片器: 視覺化的互動式篩選工具,提升儀表板的易用性
  • 時程表: 專為日期欄位設計的時間軸篩選器,支援日/月/季/年切換
  • Power Pivot: 處理大量資料、建立多表關聯、使用 DAX 進行進階計算
  • 動態儀表板: 結合樞紐分析表、切片器、時程表與圖表,打造專業的商業智慧工具
  • DAX 公式: 學習基礎 DAX 語法,為未來的 Power BI 學習打下基礎
