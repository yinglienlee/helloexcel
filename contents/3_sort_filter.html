📊 資料排序與篩選

靈活運用單層次、多層次排序與自動/進階篩選技巧

1資料排序基礎

排序是將資料按照特定順序重新排列的功能,是資料分析的第一步。

🔹 單層次排序

針對單一欄位進行遞增或遞減排序。

操作步驟:

  1. 點選要排序的欄位中的任一儲存格
  2. 點選「資料」選項卡
  3. 選擇「排序與篩選」群組中的:
    • 🔼 從最小到最大排序(遞增排序)
    • 🔽 從最大到最小排序(遞減排序)
💡 快速排序技巧:
  • 數值欄位:最小到最大(1→9)或最大到最小(9→1)
  • 文字欄位:A→Z 或 Z→A 按字母順序排序
  • 日期欄位:從最早到最新或從最新到最早
  • 不需要選取整個資料範圍,Excel 會自動識別連續的資料區域

🔹 多層次排序

當需要依據多個條件排序時使用,例如先按部門排序,再按薪資排序。

操作步驟:

  1. 選取資料範圍內的任一儲存格
  2. 點選「資料」→「排序」(或按快速鍵 Alt+D+S)
  3. 在排序對話框中:
    • 設定主要排序依據(第一優先)
    • 點選「新增層級」增加次要排序依據
    • 可繼續新增更多排序層級
  4. 選擇每個層級的排序方式(遞增/遞減)
  5. 點選「確定」執行排序

  • Excel 無法直接「復原排序」,使用 Ctrl+Z 可復原

2自動篩選功能

篩選可以暫時隱藏不符合條件的資料,只顯示符合特定條件的記錄,而不改變原始資料順序。

🔹 啟用自動篩選

操作步驟:

  1. 點選資料範圍內的任一儲存格
  2. 點選「資料」→「篩選」(或按快速鍵 Ctrl+Shift+L)
  3. 標題列的每個欄位右側會出現下拉箭頭 ▼

🔹 篩選類型與操作

📋 值篩選(勾選清單)

最簡單直接的篩選方式,適合類別資料。

  • 點選欄位的下拉箭頭
  • 取消勾選「全選」
  • 只勾選要顯示的項目
  • 點選「確定」

範例:篩選「業務部」的員工

🔢 數字篩選

針對數值欄位的條件篩選。

  • 等於 / 不等於
  • 大於 / 大於或等於
  • 小於 / 小於或等於
  • 介於(設定範圍)
  • 前 10 項(前幾名/後幾名)
  • 高於/低於平均值

📝 文字篩選

針對文字欄位的條件篩選。

  • 等於 / 不等於
  • 開頭為 / 結尾為
  • 包含 / 不包含
  • 自訂篩選(進階條件)

範例:篩選姓名包含「王」的員工

📅 日期篩選

針對日期欄位的特殊篩選。

  • 今天 / 昨天 / 明天
  • 本週 / 上週 / 下週
  • 本月 / 上月 / 下月
  • 本季 / 本年
  • 自訂日期範圍

🔹 多重篩選條件

可以同時對多個欄位設定篩選條件,Excel 會顯示同時符合所有條件的記錄(AND 邏輯)。

範例:找出「業務部」且「薪資大於 50,000」的員工 步驟: 1. 在「部門」欄位篩選 → 只勾選「業務部」 2. 在「薪資」欄位篩選 → 數字篩選 → 大於 → 輸入 50000 3. 結果:只顯示同時符合兩個條件的記錄
💡 篩選狀態識別:
  • 有篩選條件的欄位,下拉箭頭會變成漏斗圖示 🔽
  • 列號會變成藍色,表示有隱藏的列
  • 左下角狀態列會顯示「找到 X 筆記錄(總共 Y 筆)」
  • 清除單一欄位篩選:點選該欄位下拉箭頭 → 清除篩選
  • 清除所有篩選:資料 → 清除(或快速鍵 Alt+D+C)

3進階篩選

進階篩選可以設定更複雜的條件組合(AND 和 OR 邏輯),並可將篩選結果複製到其他位置。

🔹 進階篩選 vs 自動篩選

比較項目 自動篩選 進階篩選
條件複雜度 簡單條件,同一欄位只能設定一種條件 可設定複雜的 AND/OR 條件組合
篩選位置 只能在原位置顯示結果 可複製到其他位置,保留原始資料
條件設定 直接在下拉選單中設定 需在儲存格範圍中建立條件表
適用場景 快速查看、簡單篩選 複雜條件、產生報表、資料提取

🔹 進階篩選操作步驟

步驟 1:準備條件範圍

在資料表上方或其他位置建立條件表:

  1. 複製資料表的標題列(欄位名稱必須完全相同)
  2. 在標題列下方輸入篩選條件
  3. 同一列的多個條件為 AND(且)關係
  4. 不同列的條件為 OR(或)關係
條件範圍範例:篩選條件表
員工編號 姓名 部門 職位 薪資
業務部 >50000
研發部 經理
上述條件表的意義: 第一列條件:部門 = 業務部 AND 薪資 > 50000 第二列條件:部門 = 研發部 AND 職位 = 經理 兩列之間是 OR 關係,所以篩選結果為: (業務部 且 薪資>50000) 或 (研發部 且 職位=經理)

步驟 2:執行進階篩選

  1. 點選資料範圍內的任一儲存格
  2. 點選「資料」→「進階」
  3. 在進階篩選對話框中設定:
    • 方式:
      • ✓ 在原有範圍顯示篩選結果(隱藏不符合的列)
      • ✓ 將篩選結果複製到其他位置(產生新資料表)
    • 清單範圍:選取包含標題的完整資料範圍
    • 條件範圍:選取包含標題的條件表範圍
    • 複製到:(如選擇複製)指定結果要貼上的位置
    • 不重複的記錄:勾選可過濾重複資料
  4. 點選「確定」執行篩選

🔹 進階篩選條件設定技巧

條件類型 設定方式 範例
精確比對 直接輸入完整內容 業務部
數字比較 使用比較運算子 >50000、<=30000、<>0
文字包含 使用萬用字元 * *王*(包含「王」)、王*(以「王」開頭)
範圍條件 同一欄位設定兩個條件(不同列) 薪資 >30000 且 <60000
日期範圍 使用比較運算子 >=2024/1/1、<=2024/12/31
空白值 輸入 = = (篩選空白儲存格)
非空白值 輸入 <> <> (篩選非空白儲存格)
⚠️ 進階篩選常見錯誤:
  • 條件範圍標題與資料表標題不一致 → 必須完全相同(包括空格)
  • 條件範圍包含空白列 → Excel 會將空白列視為結束
  • 清單範圍未包含標題列 → 會將第一列資料當成標題
  • 複製位置與原資料重疊 → 會覆蓋原始資料

6進階技巧與最佳實務

🔧 排序與篩選的組合應用

✅ 最佳實務

  • 先篩選,再排序:縮小資料範圍後排序更有意義
  • 使用「複製到其他位置」:保留原始資料完整性
  • 定期清除篩選:避免遺漏完整資料
  • 使用表格功能(Ctrl+T):自動管理篩選範圍
  • 為條件範圍命名:便於進階篩選重複使用

❌ 常見錯誤

  • 忘記資料有篩選狀態:導致統計結果錯誤
  • 在篩選後直接刪除資料:可能誤刪隱藏的記錄
  • 排序時未選取完整範圍:資料對應錯亂
  • 條件表與資料表標題不一致:進階篩選失敗
  • 在合併儲存格上使用篩選:功能異常

🔍 疑難排解

問題 原因與解決方法
篩選下拉箭頭沒有出現 • 可能處於編輯模式,按 Esc 退出
• 工作表被保護,需解除保護
• 儲存格是合併儲存格,需取消合併
排序後資料對應錯誤 • 排序前未選取完整範圍
• 資料中有空白列或欄
• 使用 Ctrl+Z 復原,重新選取完整範圍排序
進階篩選沒有結果 • 條件表標題與資料表不一致
• 條件設定錯誤(如忘記輸入比較運算子)
• 條件範圍包含多餘的空白列
數字篩選選項無法使用 • 該欄位儲存為文字格式
• 選取欄位 → 資料 → 分欄 → 完成(轉換為數字)
日期篩選顯示不正確 • 日期儲存為文字格式
• 檢查儲存格格式,確保設定為「日期」格式
