📊 圖表基礎與設計原則

掌握數據視覺化技巧,讓數據說話更有力

1圖表類型選擇指南

選擇正確的圖表類型是數據視覺化的第一步。不同的圖表類型適合呈現不同性質的數據。

📊 柱狀圖 (Column/Bar Chart)

▮▮▮

適用情境:

  • 比較不同類別的數值
  • 顯示數量差異
  • 時間序列的離散數據

常見應用:

  • 各部門銷售額比較
  • 不同產品的產量
  • 年度預算執行情況

📈 折線圖 (Line Chart)

╱╲╱

適用情境:

  • 顯示趨勢變化
  • 連續時間序列數據
  • 多組數據的趨勢比較

常見應用:

  • 股價走勢圖
  • 溫度變化曲線
  • 人口增長趨勢

🥧 圓餅圖 (Pie Chart)

◐

適用情境:

  • 顯示部分與整體關係
  • 百分比構成
  • 類別不超過 5-7 個

常見應用:

  • 市場佔有率
  • 預算分配比例
  • 成本結構分析

📉 散佈圖 (Scatter Chart)

∴∵∴

適用情境:

  • 顯示兩變數間的關係
  • 相關性分析
  • 異常值識別

常見應用:

  • 身高與體重關係
  • 廣告支出與銷售額
  • 溫度與能源消耗
💡 圖表選擇原則:
  • 比較數值 → 柱狀圖
  • 顯示趨勢 → 折線圖
  • 呈現占比 → 圓餅圖
  • 分析相關性 → 散佈圖

2圖表元件與構成

了解圖表的各個元件,才能精確控制圖表的呈現效果。

圖表標題 說明圖表主題的文字,應簡潔明確。
位置: 通常放在圖表上方
設定方式: 圖表工具 → 圖表設計 → 新增圖表項目 → 圖表標題
座標軸標題 說明 X 軸(水平軸)和 Y 軸(垂直軸)代表的變數與單位。
X 軸: 通常代表類別或時間
Y 軸: 通常代表數值或測量值
重要性: 必須標註單位(如:元、件、%)
圖例 (Legend) 解釋圖表中不同顏色或符號代表的數據系列。
位置: 可放置在上、下、左、右
建議: 數據系列少時可放右側,多時可放底部
數據標籤 直接在圖表上顯示數值,提高可讀性。
格式: 可顯示數值、百分比或類別名稱
注意: 數據點太多時會顯得擁擠
格線 輔助讀取數值的參考線。
主要格線: 對應主要刻度
次要格線: 對應次要刻度
建議: 不宜過多,以免干擾視覺
趨勢線 顯示數據的整體走勢,用於預測分析。
類型: 線性、指數、對數、多項式等
應用: 銷售預測、成長率分析
📖 專有名詞解釋:

• 數據系列 (Data Series): 圖表中的一組相關數據點,通常用相同顏色或符號表示。例如在比較多個產品銷售額的圖表中,每個產品就是一個數據系列。

• 趨勢線 (Trendline): 根據現有數據點繪製的曲線,用來顯示數據的整體趨勢方向,可用於預測未來走向。常見類型包括線性(直線)、指數(加速成長)、對數(減速成長)等。

• 座標軸刻度 (Axis Scale): 座標軸上的數值間隔。主要刻度間隔較大,次要刻度間隔較小。適當的刻度設定能讓圖表更易讀。

3圖表建立步驟

基本操作流程

  1. 選取數據範圍: 選取包含標題列和數據列的完整範圍
  2. 插入圖表: 點選「插入」分頁 → 選擇圖表類型 → 選擇子類型
  3. 調整圖表位置: 拖曳圖表到適當位置,調整大小
  4. 編輯圖表標題: 點選標題文字直接修改
  5. 新增軸標題: 圖表工具 → 新增圖表項目 → 座標軸標題
  6. 設定數據標籤: 圖表工具 → 新增圖表項目 → 資料標籤
  7. 格式美化: 調整顏色、字型、樣式等

快速鍵技巧

  • Alt + F1: 在工作表中建立預設圖表
  • F11: 在新工作表中建立圖表
  • Ctrl + 1: 開啟格式設定窗格
⚠️ 常見錯誤:
  • 選取數據範圍不完整,導致圖表缺少資訊
  • 使用不適當的圖表類型,無法清楚呈現數據特性
  • 圖表標題或軸標題過於簡略,讀者無法理解
  • 忘記標註單位,造成數值誤解
  • 使用過多格線或裝飾,干擾數據呈現

4圖表格式設定技巧

顏色配置原則

對比清晰

使用對比色區分不同數據系列,避免相近顏色造成混淆。

色彩一致

同一主題的圖表使用一致的配色方案,建立視覺識別。

避免過度

不要使用過多顏色,一般建議不超過 5-7 種主色。

考慮列印

如需列印,確保黑白列印時仍能區分不同數據。

字型與大小設定

  • 標題: 14-18 pt,粗體,置中對齊
  • 軸標題: 11-12 pt,粗體
  • 軸標籤: 9-10 pt,一般
  • 數據標籤: 8-9 pt,視圖表大小調整
  • 圖例: 9-10 pt,一般

趨勢線新增方法

  1. 在圖表上點選數據系列(點或線)
  2. 按右鍵選擇「新增趨勢線」
  3. 在右側窗格選擇趨勢線類型:
    • 線性: 數據呈直線趨勢
    • 指數: 數據呈加速成長
    • 對數: 數據呈減速成長
    • 多項式: 數據有起伏波動
    • 移動平均: 平滑短期波動
  4. 勾選「顯示方程式」可顯示趨勢線公式
  5. 勾選「顯示 R 平方值」可顯示擬合度
📖 專有名詞解釋:

• R 平方值 (R-squared): 又稱決定係數,用來衡量趨勢線與實際數據的擬合程度,範圍從 0 到 1。R² = 1 表示完美擬合,R² = 0 表示完全不相關。一般認為 R² > 0.7 代表擬合度良好。

• 移動平均 (Moving Average): 取連續若干期數據的平均值,用來平滑短期波動,突顯長期趨勢。常用於股價分析、銷售預測等。

• 多項式趨勢線 (Polynomial Trendline): 使用多項式方程式擬合數據,適合有多個轉折點的數據。階數越高曲線越彎曲,但階數過高可能造成過度擬合。

5專業圖表設計原則

📋 CLEAR 設計法則

C - Clear
清晰明確		 • 圖表目的一目了然
• 標題直接點出重點
• 避免不必要的裝飾元素
• 數據標示清楚易讀
L - Logical
邏輯合理		 • 選擇適合的圖表類型
• 座標軸刻度合理
• 數據排序有意義(如由大到小)
• 時間序列保持正確順序
E - Economical
精簡有效		 • 移除不必要的格線
• 避免 3D 效果(會扭曲視覺)
• 減少圖例項目(可直接標註)
• 一張圖表傳達一個重點
A - Accurate
精確誠實		 • Y 軸從 0 開始(除非有特殊理由)
• 不扭曲比例關係
• 完整呈現數據,不隱藏不利資訊
• 標註數據來源與時間
R - Readable
易於閱讀		 • 字體大小適中
• 顏色對比清晰
• 考慮色盲友善配色
• 圖表大小適合載體(螢幕/紙本)
✅ 專業圖表檢查清單:
  • □ 圖表標題是否清楚說明主題?
  • □ X 軸和 Y 軸都有標題和單位嗎?
  • □ 圖例位置是否適當?
  • □ 顏色選擇是否清晰易辨?
  • □ 是否移除了不必要的格線和裝飾?
  • □ 數據標籤是否清楚易讀?
  • □ Y 軸比例是否合理?(一般從 0 開始)
  • □ 圖表能否在 5 秒內理解重點?

6實務範例:人口增長趨勢分析

以下示範如何製作一個專業的人口增長趨勢圖,這是社會科學研究中常見的視覺化需求。

範例數據:台灣人口變化(千人)

年份 總人口 出生人數 死亡人數 自然增加率(%)
2015 23,492 213 163 0.21
2016 23,539 208 172 0.15
2017 23,571 193 171 0.09
2018 23,589 181 173 0.03
2019 23,603 177 176 0.00
2020 23,561 165 173 -0.03
2021 23,375 154 183 -0.12
2022 23,264 138 207 -0.30
📌 人口增加率小知識：

人口增加率（Natural Increase Rate）用來衡量一個地區人口「自然變動」的情況， 只考慮出生與死亡，不包含移民因素。

計算公式：
(出生人數 − 死亡人數) ÷ 總人口 × 100%

判讀方式：
  • 📈 正值：出生多於死亡，人口自然成長
  • ➖ 0：出生與死亡人數相近，人口持平
  • 📉 負值：死亡多於出生，人口自然減少

製作步驟

  1. 選取數據: 選取 A1:B9(年份和總人口)
  2. 插入折線圖: 插入 → 折線圖 → 含有資料標記的折線圖
  3. 設定標題: 點選標題改為「台灣人口變化趨勢(2015-2022)」
  4. 新增軸標題:
    • Y 軸:「人口數(千人)」
    • X 軸:「年份」
  5. 新增趨勢線: 點選數據系列 → 右鍵 → 新增趨勢線 → 線性
  6. 勾選選項: 顯示方程式、顯示 R 平方值
  7. 美化格式: 調整顏色、線條粗細、標記大小
💡 分析要點:
  • 從折線圖可清楚看出人口在 2020 年達到高峰後開始下降
  • 趨勢線顯示整體呈現緩慢上升後下降的趨勢
  • 可搭配「自然增加率」製作雙軸圖表,分析人口變化原因
  • 建議使用柱狀圖呈現出生與死亡人數的對比

7實務範例:成本結構分析

圓餅圖最適合呈現成本結構與預算分配,以下為某製造業的成本分析範例。

範例數據:製造成本結構

成本項目 金額(萬元) 占比
原物料成本 4,500 45%
人力成本 2,800 28%
製造費用 1,500 15%
折舊攤提 700 7%
其他費用 500 5%
合計 10,000 100%

製作步驟

  1. 選取數據: 選取 A1:B6(成本項目和金額,不含合計列)
  2. 插入圓餅圖: 插入 → 圓形圖或環圈圖 → 圓形圖
  3. 設定標題:「2024 年製造成本結構分析」
  4. 新增數據標籤: 圖表工具 → 新增圖表項目 → 資料標籤 → 更多選項
    • 勾選「類別名稱」
    • 勾選「百分比」
    • 取消勾選「值」(避免重複)
  5. 調整圖例: 考慮移除圖例(因為數據標籤已包含類別名稱)
  6. 突顯重點: 點選「原物料成本」扇形 → 右鍵 → 格式化資料點 → 爆炸圖扇區偏移 10%
  7. 顏色配置: 使用商務配色,避免過於鮮豔
⚠️ 圓餅圖使用注意事項:
  • 類別不宜超過 5-7 個,過多會難以閱讀
  • 避免使用 3D 圓餅圖,會扭曲視覺比例
  • 將最重要的類別放在 12 點鐘方向(Excel 會自動排序,可手動調整)
  • 如果有多個小比例項目,可合併為「其他」類別
  • 當需要比較多個圓餅圖時,建議改用堆疊柱狀圖

📝 課後練習題

練習題 1:地區銷售業績比較圖

任務描述:

某連鎖零售公司需要分析各地區分店的季度銷售表現,請製作適當的圖表呈現以下數據:

地區 Q1 銷售額 Q2 銷售額 Q3 銷售額 Q4 銷售額 全年總計
北部 12,500 13,800 15,200 18,600 (公式)
中部 8,600 9,200 10,500 11,800 (公式)
南部 9,800 10,500 11,200 13,500 (公式)
東部 4,200 4,800 5,500 6,200 (公式)

製作要求:

  • 圖表 1 - 群組直條圖: 比較四個地區在各季度的銷售額
    • X 軸:季度(Q1-Q4)
    • Y 軸:銷售額(千元)
    • 使用不同顏色區分地區
    • 標題:「2024 年各地區季度銷售額比較」
  • 圖表 2 - 圓餅圖: 呈現各地區全年銷售額占比
    • 使用全年總計數據
    • 顯示百分比和類別名稱
    • 標題:「2024 年各地區銷售額占比」

分析任務:

  • 哪個地區的銷售額最高?成長趨勢如何?
  • 哪個季度整體銷售表現最好?
  • 如果要提升整體業績,應該重點關注哪個地區?

進階挑戰:

  • 製作折線圖顯示各地區的季度成長趨勢
  • 計算每個地區的季度平均成長率
  • 在折線圖上新增線性趨勢線,觀察長期走向

某班級進行期末考試,教師需要分析成績分布情況與各科目表現,數據如下:

額外數據:各科平均分數

科目 本班平均 全年級平均
國文76.574.2

數學68.370.1

製作要求:

  • 圖表 1 - 堆疊橫條圖: 顯示各科目的分數分布
    • Y 軸:科目名稱
    • X 軸:學生人數
    • 使用不同顏色表示分數區間
    • 標題:「各科目成績分布情況」
  • 圖表 2 - 群組直條圖: 比較本班與全年級平均分數

  • 在比較圖中,為低於全年級平均的科目加上醒目標示
  • 新增數據標籤,顯示精確數值
💡 教學提示:堆疊橫條圖特別適合顯示組成結構,而群組直條圖適合多組數據的比較。這個練習結合了兩種圖表,能幫助你理解如何選擇最適當的視覺化方式。

練習題 3:產品生命週期與市場佔有率分析(綜合應用)

任務描述:

某科技公司推出三款產品,需要分析產品生命週期與市場競爭情況:

📖 專有名詞解釋:

• 產品生命週期 (Product Life Cycle): 產品從推出到退出市場的過程,通常分為導入期、成長期、成熟期、衰退期四個階段。不同階段有不同的銷售特徵和市場策略。

• 市場佔有率 (Market Share): 某產品或公司的銷售額占整體市場銷售額的百分比,是衡量市場競爭力的重要指標。

• 散佈圖氣泡大小: 在散佈圖中,除了 X、Y 座標外,還可以用氣泡大小表示第三個變數,形成三維數據視覺化。

    • 哪款產品處於成長期?哪款進入成熟期?哪款開始衰退?
    • 產品 A 的銷售高峰出現在第幾個月?
    • 根據趨勢線,預測未來三個月各產品的銷售走向
  • 從氣泡圖分析:
    • 哪款產品最具成長潛力?(考慮上市時間與市佔率)
    • 與競爭者 X 相比,我司產品的競爭優勢與劣勢?
    • 如果要追上市場領導者,應該採取什麼策略?

格式與設計要求:

  • 折線圖使用不同線條樣式(實線、虛線、點線)區分產品
  • 趨勢線使用較淺的顏色,避免干擾實際數據
  • 氣泡圖中為「競爭者 X」使用特殊顏色標示(如紅色)
  • 在氣泡圖上新增數據標籤,顯示產品名稱
  • 兩張圖表使用一致的配色方案
  • 新增文字方塊標註重要發現(如「產品 A 在第 8 月達到銷售高峰」)

進階挑戰:

  • 計算每款產品的月均成長率,判斷成長動能
  • 製作組合圖(折線+柱狀)顯示銷售量與市佔率的關係
  • 使用條件格式在數據表中標示銷售量>50的儲存格
  • 撰寫簡短的市場分析報告(200字),整合兩張圖表的洞察
💡 實務應用提示:這個綜合練習模擬了企業產品經理的實際工作場景。在真實商業環境中,產品經理需要同時監控多個產品的表現,並根據數據做出策略決策。掌握這種多維度數據視覺化能力,是數據驅動決策的關鍵技能!

🎓 學習重點回顧

  • 圖表類型選擇: 比較用柱狀圖、趨勢用折線圖、占比用圓餅圖、相關性用散佈圖
  • 圖表元件: 標題、軸標題、圖例、數據標籤、格線、趨勢線缺一不可
  • CLEAR 原則: 清晰、邏輯、精簡、精確、易讀是專業圖表的五大要素
  • 趨勢線應用: 線性、指數、對數、多項式各有適用場景,R² 值反映擬合度
  • 設計美學: 適當的配色、字型、留白能大幅提升圖表專業度
  • 避免誤導: Y 軸從 0 開始、避免 3D 效果、不扭曲比例是誠實呈現數據的基本原則
