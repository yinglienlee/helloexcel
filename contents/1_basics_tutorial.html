📊 Excel 環境介紹與基礎操作

從零開始熟悉 Excel 介面、儲存格操作與基本資料輸入

1Excel 介面認識

A. 🎯 功能區 (Ribbon)

包含「常用」、「插入」、「版面配置」、「公式」、「資料」、「校閱」、「檢視」等分頁，集中所有操作工具。

B. 📝 名稱方塊

顯示目前選取的儲存格位置（如 A1、B5），也可以直接輸入儲存格位址快速跳轉。

C. ✏️ 資料編輯列

顯示與編輯儲存格內容，支援公式輸入與編輯。

D. 📋 工作表標籤

位於視窗底部，用於切換不同工作表，預設為 Sheet1、Sheet2 等。

E. 🔤 欄標題 (A, B, C...)

英文字母標示，最多可到 XFD（16,384 欄）。

F. 🔢 列標題 (1, 2, 3...)

數字標示，最多可到 1,048,576 列。

💡 專業提示：儲存格位址由「欄標題+列標題」組成，例如 B5 表示 B 欄第 5 列的儲存格。

2儲存格與工作表基本操作

儲存格選取方式

  • 單一儲存格：直接點擊
  • 連續範圍：按住滑鼠左鍵拖曳，或點擊起始儲存格後按 Shift+點擊結束儲存格
  • 不連續範圍：按住 Ctrl 鍵點擊多個儲存格
  • 整欄/整列：點擊欄標題或列標題
  • 全選：點擊左上角的全選按鈕或按 Ctrl+A

工作表管理

  • 新增工作表：點擊工作表標籤旁的 ➕ 圖示
  • 重新命名：雙擊工作表標籤名稱
  • 刪除工作表：右鍵點擊工作表標籤 → 刪除
  • 移動工作表：拖曳工作表標籤到新位置
  • 複製工作表：按住 Ctrl 鍵拖曳工作表標籤

3基本資料輸入與儲存格格式

資料類型與輸入技巧

數值格式 直接輸入數字，Excel 會自動右對齊。
例如：1000、-500、3.14
設定方式：常用 → 數值 → 選擇格式（數值、貨幣、會計等）
日期格式 輸入格式：2024/1/15 或 2024-1-15
Excel 會自動識別並轉換為日期格式
設定方式：常用 → 數值 → 簡短日期/完整日期
百分比格式 輸入 0.85 後設定為百分比會顯示 85%
或直接輸入 85% 也可以
設定方式：常用 → 數值 → % (百分比符號)
文字格式 如果數字要當文字（如員工編號 001），
在數字前加上單引號 '001
設定方式：常用 → 數值 → 文字
⚠️ 常見錯誤：輸入日期時若顯示為數字（如 45674），表示格式設定錯誤，需要重新設定為日期格式。

4相對參照與絕對參照

這是 Excel 中最重要的概念之一，影響公式複製時的行為。

📌 相對參照 (A1)

當公式被複製到其他儲存格時，參照會自動調整。

範例：在 C2 輸入公式 =A2*B2 向下複製到 C3 時，公式自動變為 =A3*B3 向下複製到 C4 時，公式自動變為 =A4*B4

📌 絕對參照 ($A$1)

當公式被複製時，參照保持不變，欄和列都固定。

範例：在 C2 輸入公式 =A2*$B$1 向下複製到 C3 時，公式變為 =A3*$B$1 向下複製到 C4 時，公式變為 =A4*$B$1 （$B$1 始終指向 B1 儲存格）

📌 混合參照 ($A1 或 A$1)

  • $A1：欄固定，列可變動
  • A$1：列固定，欄可變動
💡 快速鍵：選取儲存格參照後按 F4 鍵，可在相對、絕對、混合參照之間快速切換。
• 按一次：$A$1 (絕對參照)
• 按兩次：A$1 (列固定)
• 按三次：$A1 (欄固定)
• 按四次：A1 (相對參照)

公式應用說明

  • 總成本 (E 欄)：=C2*D2 (生產數量 × 單位成本)
  • 良品率 (F 欄)：指生產過程中合格產品佔總生產數量的百分比，是品質管理的重要指標
  • 不良品數 (G 欄)：=C2*(1-F2) (生產數量 × (1-良品率))
  • 若要計算總生產成本：=SUM(E2:E6)
  • 若要計算平均良品率：=AVERAGE(F2:F6)

格式設定重點

  1. 日期欄 (A 欄)：設定為「簡短日期」格式
  2. 數量欄 (C 欄)：設定為「數值」格式，千位分隔符號
  3. 成本欄 (D、E 欄)：設定為「貨幣」或「數值」格式，保留兩位小數
  4. 良品率 (F 欄)：設定為「百分比」格式，保留一位小數
  5. 標題列：加粗、置中對齊、背景色填滿

• 本位幣（Base Currency）：企業記帳使用的主要貨幣，台灣公司通常使用台幣（TWD）作為本位幣。

• 換算金額（Converted Amount）：外幣金額按匯率換算成本位幣的金額。計算公式：換算金額 = 外幣金額 × 匯率。

🎓 學習重點回顧

  • 相對參照 (A1)：複製公式時自動調整，適合一般計算

  • 混合參照 ($A1 或 A$1)：部分固定，適合特殊表格結構
  • 快速鍵 F4：快速在不同參照類型間切換
  • 格式設定：讓數據更專業、更易讀
  • IF 函數：根據條件做出判斷，是商業邏輯的基礎
  • SUM、AVERAGE：基本統計函數，廣泛應用於各種分析
