1條件統計函數家族

📌 單條件函數：COUNTIF、SUMIF、AVERAGEIF

這三個函數允許我們根據單一條件來篩選資料並進行統計運算。

COUNTIF

功能：計算符合條件的儲存格數量

=COUNTIF(範圍, 條件)

範例：

=COUNTIF(A2:A10, "台北") 計算 A2:A10 範圍內等於「台北」的數量

SUMIF

功能：加總符合條件的數值

=SUMIF(條件範圍, 條件, 加總範圍)

範例：

=SUMIF(B2:B10, ">80", C2:C10) 當 B 欄 > 80 時，加總對應的 C 欄數值

AVERAGEIF

功能：計算符合條件的平均值

=AVERAGEIF(條件範圍, 條件, 平均範圍)

範例：

=AVERAGEIF(A2:A10, "女", B2:B10) 計算 A 欄為「女」時，B 欄的平均值
💡 專有名詞解釋：
  • 條件 (Criteria)：用於篩選資料的規則，可以是文字、數字、比較運算符 (如 ">80"、"<=100") 或萬用字元 (如 "*台北*")。
  • 範圍 (Range)：要進行檢查或運算的儲存格區域,例如 A2:A10。
  • 萬用字元 (Wildcard)： 使用 * 代表任意字元,例如 "台*" 可以匹配「台北」、「台中」、「台南」。

📌 多條件函數：COUNTIFS、SUMIFS、AVERAGEIFS

當需要同時滿足多個條件時,使用這些「S」家族函數。所有條件都必須為 TRUE (邏輯 AND)。

COUNTIFS

功能：計算符合多個條件的數量

=COUNTIFS(條件範圍1, 條件1, 條件範圍2, 條件2, ...)

範例：

=COUNTIFS(A2:A10, "台北", B2:B10, ">80") 計算 A 欄=台北 且 B 欄>80 的數量

SUMIFS

功能：加總符合多個條件的數值

=SUMIFS(加總範圍, 條件範圍1, 條件1, 條件範圍2, 條件2, ...)

範例：

=SUMIFS(C2:C10, A2:A10, "台北", B2:B10, ">=60") 計算 A 欄=台北 且 B 欄>=60 時的 C 欄總和

AVERAGEIFS

功能：計算符合多個條件的平均值

=AVERAGEIFS(平均範圍, 條件範圍1, 條件1, 條件範圍2, 條件2, ...)

範例：

=AVERAGEIFS(D2:D10, A2:A10, "女", B2:B10, ">25") 計算 A 欄=女 且 B 欄>25 時的 D 欄平均
⚠️ 注意事項：
  • 參數順序差異：SUMIF/AVERAGEIF 的「加總/平均範圍」在最後,但 SUMIFS/AVERAGEIFS 的對應範圍在最前面!
  • 條件格式：數字比較需用雙引號,例如 ">80"、"<=100"。
  • 文字匹配：預設不區分大小寫,但可使用萬用字元做模糊匹配。

📊 實務範例：社會科學態度調查統計

某社會學研究調查了民眾對環保政策的態度,包含受訪者的地區、年齡、教育程度與態度評分 (1-10 分)。

編號 (A) 地區 (B) 年齡 (C) 教育程度 (D) 態度評分 (E)
1台北28大學8
2台中45高中6
3台北35碩士9
4高雄52大學7
5台北23大學8
6台中38碩士9
7高雄41高中5
8台北29碩士10

統計分析問題：

  1. 計算台北地區的受訪者數量：
    =COUNTIF(B2:B9, "台北")
  2. 計算年齡大於 30 歲的受訪者總態度評分：
    =SUMIF(C2:C9, ">30", E2:E9)
  3. 計算大學學歷者的平均態度評分：
    =AVERAGEIF(D2:D9, "大學", E2:E9)
  4. 計算台北地區且年齡小於 30 歲的受訪者數量：
    =COUNTIFS(B2:B9, "台北", C2:C9, "<30")
  5. 計算碩士學歷且態度評分 >= 8的總評分：
    =SUMIFS(E2:E9, D2:D9, "碩士", E2:E9, ">=8")
  6. 計算台北地區且碩士學歷者的平均態度評分：
    =AVERAGEIFS(E2:E9, B2:B9, "台北", D2:D9, "碩士")
💡 社會科學應用情境：
  • 交叉分析 (Cross-tabulation)：使用 COUNTIFS 進行多變量交叉統計,例如分析「性別 × 年齡層 × 政黨傾向」的分布。
  • 分層抽樣檢驗：使用 AVERAGEIFS 計算特定群體的平均值,檢驗樣本代表性。
  • 態度量表分析：結合條件函數統計不同人口變項下的李克特量表 (Likert Scale) 分數。

2規劃求解 (Solver)

📌 什麼是 Solver?

Solver 是 Excel 的增益集工具,用於解決最佳化問題 (Optimization Problems)。它可以在給定的限制條件下,找出使目標函數達到最大值、最小值或特定值的變數組合。

💡 專有名詞解釋：
  • 線性規劃 (Linear Programming, LP)：一種數學優化方法,目標函數和限制條件都是線性關係。常用於資源分配、生產排程、運輸問題等。
  • 目標函數 (Objective Function)：要最大化或最小化的數學式,例如「總利潤」、「總成本」。
  • 決策變數 (Decision Variables)：可以調整的變數,例如「生產 A 產品的數量」、「投資金額」。
  • 限制條件 (Constraints)：決策變數必須滿足的條件,例如「總預算 ≤ 100 萬」、「產量 ≥ 0」。

📌 啟用 Solver

  1. 點選 Excel 功能表的「檔案」→「選項」
  2. 選擇「增益集」
  3. 在「管理」下拉選單選擇「Excel 增益集」,點選「執行」
  4. 勾選「Solver 增益集」,按「確定」
  5. 完成後,在「資料」功能區會出現「規劃求解」按鈕

📊 Solver 操作步驟

步驟 1：設定目標

指定要最大化、最小化或達到特定值的儲存格 (目標函數)。

步驟 2：設定變數儲存格

指定可以改變的儲存格範圍 (決策變數)。

步驟 3：加入限制條件

設定決策變數必須滿足的條件,例如 ≥ 0、總和 ≤ 某值等。

步驟 4：選擇求解方法

線性規劃問題選擇「單純形 LP」。

步驟 5：執行求解

按下「求解」按鈕,等待 Solver 計算最佳解。

步驟 6：檢視結果

Solver 會將最佳解填入變數儲存格,並顯示目標值。

📊 實務範例：資源分配最佳化問題 (IE/Econ)

問題情境：
某製造公司生產 A、B 兩種產品。每生產一單位 A 需要 2 小時機器時間、3 單位原料,利潤為 $500;每生產一單位 B 需要 1 小時機器時間、4 單位原料,利潤為 $300。公司每日有 100 小時機器時間、240 單位原料可用。請問應該如何分配生產數量以最大化總利潤?

💡 問題建模：
  • 決策變數：x = A 產品生產數量, y = B 產品生產數量
  • 目標函數：最大化總利潤 = 500x + 300y
  • 限制條件：
    • 機器時間：2x + 1y ≤ 100
    • 原料限制：3x + 4y ≤ 240
    • 非負限制：x ≥ 0, y ≥ 0

Excel 工作表設計：

項目 A 產品 B 產品 限制 使用量
生產數量 (決策變數)0 (B2)0 (C2)——
單位利潤500300——
機器時間 (小時/單位)21100=2*B2+1*C2 (E4)
原料 (單位/單位)34240=3*B2+4*C2 (E5)
總利潤 (目標)=500*B2+300*C2 (B6)——

Solver 設定：

  1. 目標儲存格：B6 (總利潤),選擇「最大值」
  2. 變數儲存格：B2:C2 (A、B 產品生產數量)
  3. 限制條件：
    • E4 ≤ D4 (機器時間使用量 ≤ 100)
    • E5 ≤ D5 (原料使用量 ≤ 240)
    • B2:C2 ≥ 0 (生產數量非負)
  4. 求解方法：單純形 LP
  5. 求解結果：A 產品生產 20 單位,B 產品生產 60 單位,最大利潤為 $28,000
💡 產業應用情境：
  • 工業工程 (IE)：生產排程最佳化、設施佈置規劃、庫存管理、人力資源分配。
  • 經濟學 (Economics)：投資組合最佳化、成本最小化、效用最大化、均衡分析。
  • 供應鏈管理：運輸路線規劃、配送中心選址、訂單分配。
  • 財務金融：資產配置、風險管理、資本預算決策。
⚠️ Solver 使用注意事項：
  • 線性 vs 非線性：只有當目標函數和限制條件都是線性時,才能使用「單純形 LP」方法。非線性問題需使用 GRG 非線性或演化式方法。
  • 可行解存在性：如果限制條件互相衝突,Solver 可能找不到可行解。
  • 整數限制：如果決策變數必須為整數 (例如生產件數),需在限制條件中設定「int」。
  • 敏感度分析：求解完成後可檢視「敏感度報告」,了解參數變動對最佳解的影響。

3條件函數家族比較表

函數 條件數量 語法 用途
COUNTIF 單一條件 =COUNTIF(範圍, 條件) 計算符合條件的儲存格數量
COUNTIFS 多條件 =COUNTIFS(範圍1, 條件1, 範圍2, 條件2, ...) 計算同時符合多個條件的數量
SUMIF 單一條件 =SUMIF(條件範圍, 條件, 加總範圍) 加總符合條件的數值
SUMIFS 多條件 =SUMIFS(加總範圍, 條件範圍1, 條件1, ...) 加總同時符合多個條件的數值
AVERAGEIF 單一條件 =AVERAGEIF(條件範圍, 條件, 平均範圍) 計算符合條件的平均值
AVERAGEIFS 多條件 =AVERAGEIFS(平均範圍, 條件範圍1, 條件1, ...) 計算同時符合多個條件的平均值
⚠️ 語法差異重點：

單條件函數 (IF) 的「運算範圍」在最後,但多條件函數 (IFS) 的「運算範圍」在最前面!

=SUMIF(條件範圍, 條件, 加總範圍) ← 加總範圍在最後 =SUMIFS(加總範圍, 條件範圍1, 條件1, ...) ← 加總範圍在最前

練習題 3：生產排程最佳化 (Solver 應用)

任務描述：

某家具製造廠生產桌子和椅子兩種產品。生產一張桌子需要 4 小時木工時間、2 小時組裝時間,利潤為 $1,200;生產一張椅子需要 2 小時木工時間、1 小時組裝時間,利潤為 $600。工廠每週有 160 小時木工時間、80 小時組裝時間。市場需求限制:桌子每週最多賣出 30 張。

💡 問題建模提示：
  • 決策變數：設 x = 桌子生產數量, y = 椅子生產數量
  • 目標函數：最大化總利潤 = 1200x + 600y
  • 限制條件：
    • 木工時間：4x + 2y ≤ 160
    • 組裝時間：2x + 1y ≤ 80
    • 市場需求：x ≤ 30
    • 非負限制：x ≥ 0, y ≥ 0

Excel 建模步驟：

  1. 建立工作表,設定決策變數儲存格 (桌子數量 B2、椅子數量 C2),初始值設為 0
  2. 設定單位利潤 (桌子 1200、椅子 600)
  3. 計算總利潤公式：=1200*B2+600*C2
  4. 計算資源使用量：
    • 木工時間使用：=4*B2+2*C2
    • 組裝時間使用：=2*B2+1*C2
  5. 開啟 Solver 設定：
    • 目標：總利潤儲存格,選擇「最大值」
    • 變數：B2:C2
    • 限制條件：木工時間 ≤ 160、組裝時間 ≤ 80、桌子 ≤ 30、兩個變數 ≥ 0
    • 求解方法：單純形 LP
  6. 執行求解並分析結果

分析問題：

  • 最佳生產組合是多少張桌子和椅子?
  • 最大總利潤是多少?
  • 哪些資源被完全使用了? (檢查限制條件的「鬆弛變數」)
  • 如果木工時間增加 10 小時,利潤會增加多少? (敏感度分析)

🎓 學習重點回顧

  • COUNTIF / SUMIF / AVERAGEIF：單一條件的計數、加總、平均。
  • COUNTIFS / SUMIFS / AVERAGEIFS：多條件統計,所有條件需同時滿足 (AND 邏輯)。
  • 語法差異：單條件函數的運算範圍在最後,多條件函數的運算範圍在最前!
  • 條件格式：比較運算需用雙引號 (如 ">80"),文字條件直接輸入或加引號。
  • Solver：用於線性規劃等最佳化問題,需設定目標、變數與限制條件。
  • 應用情境：社會科學調查分析、資源分配最佳化、生產排程、投資組合等。
