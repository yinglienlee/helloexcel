🔮 假設分析工具

What-If Analysis — 目標搜尋、資料表、情境管理器

1什麼是假設分析？

假設分析 (What-If Analysis) 是一組讓你探索「如果某個條件改變，結果會如何變化」的 Excel 工具。 在財務規劃、生產排程、定價決策等情境中，你往往需要回答以下問題：

  • 售價需要設定在多少，才能剛好達到損益平衡？
  • 生產線的機器速度如果從 80 件/小時提升到 120 件/小時，日產量會如何變化？
  • 在樂觀、普通、悲觀三種情境下，公司的淨利分別是多少？

Excel 提供三種假設分析工具，分別對應不同的使用情境：

🎯 目標搜尋 (Goal Seek)

已知目標結果，反推所需的輸入值。
例：利潤要達到 $50,000，售價應設為多少？

📊 資料表 (Data Table)

一次性計算一個或兩個變數在多個數值下的所有結果，形成完整的敏感性分析矩陣。

📋 情境管理器 (Scenario Manager)

儲存多組假設條件（如樂觀／普通／悲觀），並快速切換或產生比較摘要報表。

💡 如何開啟假設分析工具？
三種工具都位於同一個位置：
點擊「資料 (Data)」分頁 → 在「預測 (Forecast)」群組中，點擊「模擬分析 (What-If Analysis)」下拉按鈕，即可看到三種工具的選項。

2目標搜尋 (Goal Seek)

目標搜尋解決「逆向計算」問題：你已經知道希望得到的目標值， 並想讓 Excel 幫你找出某個輸入儲存格需要調整到多少，才能達成目標。

💡 專有名詞解釋
  • 設定目標儲存格 (Set Cell)：包含公式的儲存格，其計算結果就是你想要達到的目標。例如「利潤」儲存格。
  • 目標值 (To Value)：你希望目標儲存格達到的數字。例如輸入 0 代表剛好損益平衡。
  • 變數儲存格 (By Changing Cell)：Excel 會調整這個儲存格的數值，直到目標儲存格達到目標值。例如「售價」儲存格。

使用條件與限制

  • 目標搜尋每次只能調整一個變數（若需調整多個變數，請使用「規劃求解 Solver」增益集）。
  • 目標儲存格必須包含公式，且公式必須直接或間接參照到變數儲存格。
  • 目標搜尋使用數值迭代法逼近解答，結果為近似值（通常精度已足夠）。

實務範例：損益平衡點分析

某公司推出新產品，固定成本為 $120,000，每單位變動成本 $45，目前售價 $80。 問：需銷售多少單位才能達到損益平衡（利潤 = 0）？

儲存格 項目 數值 / 公式 說明
B1 售價（元/單位） $80 輸入值
B2 變動成本（元/單位） $45 輸入值
B3 固定成本（元） $120,000 輸入值
B4 銷售數量（單位） ??? ← 要求解的值 變數儲存格（Goal Seek 將調整此值）
B5 總收入（元） =B1*B4 售價 × 數量
B6 總變動成本（元） =B2*B4 變動成本 × 數量
B7 利潤（元） =B5-B6-B3 目標儲存格（要讓此值 = 0）

操作步驟

  1. 在工作表中依上表建立資料結構，並在 B4 先輸入任意初始值（如 1000）。
  2. 點擊「資料」→「模擬分析 (What-If Analysis)」→「目標搜尋 (Goal Seek)」。
  3. 在對話框中填入三個欄位：
    • 設定目標儲存格：點選 B7（利潤）
    • 目標值：輸入 0（損益平衡）
    • 藉由變更儲存格：點選 B4（銷售數量）
  4. 點擊「確定」，Excel 會自動迭代計算並將結果填入 B4。
  5. 確認結果視窗顯示「已找到解答」，B4 應顯示約 2,667 單位，此即損益平衡點。
💡 損益平衡點 (Break-Even Point, BEP) 說明
損益平衡點是指企業的總收入 = 總成本（即利潤為零）時的銷售量或銷售金額。 公式推導：
售價 × 數量 = 變動成本 × 數量 + 固定成本
→ 數量 = 固定成本 ÷（售價 − 變動成本）= $120,000 ÷ ($80 − $45) = 3,429 單位。
目標搜尋可快速驗算此結果，特別適合公式較複雜、無法用代數直接求解的情境。
⚠️ 注意：目標搜尋會直接覆寫儲存格數值
執行目標搜尋後，Excel 會把求得的結果直接填入變數儲存格。如果你想保留原始數值， 請在執行前按 Ctrl + Z 復原，或事先將原始值記錄在其他儲存格。

3資料表 (Data Table)

資料表（又稱「敏感性分析表」）可以同時計算一個或兩個輸入變數在多個不同數值下的所有輸出結果，讓你一眼看清各種情境的影響。

💡 專有名詞解釋
  • 單變數資料表 (One-Variable Data Table)：固定其他所有變數，只讓一個輸入值在不同數值間變化，觀察對某個結果的影響。例如：觀察不同售價對利潤的影響。
  • 雙變數資料表 (Two-Variable Data Table)：同時讓兩個輸入值在不同數值間變化，在矩陣中呈現所有組合的結果。例如：同時改變售價與銷售量，觀察各組合下的利潤。
  • 列輸入儲存格 (Row Input Cell)：資料表中，橫向（列方向）的變數所對應的原始輸入儲存格。
  • 欄輸入儲存格 (Column Input Cell)：資料表中，縱向（欄方向）的變數所對應的原始輸入儲存格。

單變數資料表：不同售價 vs. 利潤

以前一節的損益分析模型為基礎（B4 = 3,000 單位固定），觀察售價從 $60 到 $100 時利潤的變化。

資料表版面配置（欄輸入）

在試算表中建立以下結構：

儲存格 內容 說明
D1 =B7 公式儲存格，參照利潤。必須放在變數值列的右上方或同行。
C2 $60 欄方向（縱向）的變數值清單：要測試的售價。
C3$70
C4$80
C5$90
C6$100
D2:D6 （Excel 自動填入） 資料表執行後，Excel 自動計算各售價下的利潤結果。

操作步驟（單變數）

  1. 在工作表中建立上述版面：D1 輸入 =B7，C2:C6 輸入 $60、$70、$80、$90、$100。
  2. 選取整個資料表範圍：框選 C1:D6（含公式儲存格與所有變數值）。
  3. 點擊「資料」→「模擬分析」→「資料表 (Data Table)」。
  4. 因為售價清單是縱向（欄方向）排列：
    • 列輸入儲存格：留白
    • 欄輸入儲存格：點選 B1（原始售價儲存格）
  5. 點擊「確定」，D2:D6 即自動填入各售價對應的利潤。

雙變數資料表：售價 × 銷售數量 矩陣

同時讓售價（列方向：$60–$100）與銷售數量（欄方向：1,000–5,000）各自變化， 產生一個 5×5 的利潤矩陣。

1,000 件 2,000 件 3,000 件 4,000 件 5,000 件
售價 $60 −$75,000 −$30,000 $15,000 $60,000 $105,000
售價 $70 −$70,000 −$20,000 $30,000 $80,000 $130,000
售價 $80 −$65,000 −$10,000 $45,000 $100,000 $155,000
售價 $90 −$60,000 $0 $60,000 $120,000 $180,000
售價 $100 −$55,000 $10,000 $75,000 $140,000 $205,000

操作步驟（雙變數）

  1. 在交叉點（左上角）儲存格（如 G1）輸入 =B7（利潤公式）。
  2. 在同一列（G1 右側）填入列方向的變數值：$60、$70、$80、$90、$100。
  3. 在同一欄（G1 下方）填入欄方向的變數值：1,000、2,000、3,000、4,000、5,000。
  4. 選取整個矩陣範圍（含左上角公式儲存格、列標題、欄標題）。
  5. 點擊「資料」→「模擬分析」→「資料表」。
    • 列輸入儲存格：點選 B1（售價 ─ 對應列方向）
    • 欄輸入儲存格：點選 B4（銷售數量 ─ 對應欄方向）
  7. 點擊「確定」，矩陣內部自動填滿所有組合的利潤結果。
⚠️ 資料表的計算結果是陣列公式，無法單獨編輯
Excel 會以 {=TABLE(...)} 陣列公式填入結果。若需修改，必須選取整個結果範圍後再一次刪除，不能只刪除單一儲存格。

4情境管理器 (Scenario Manager)

情境管理器讓你預先定義並儲存多組假設條件組合（稱為「情境」）， 之後可以一鍵切換查看不同情境下的工作表數值，也可以產生一份摘要報表並排比較所有情境的結果。

💡 專有名詞解釋
  • 情境 (Scenario)：一組特定假設值的組合。例如「樂觀情境」可能是高售價＋高銷售量；「悲觀情境」是低售價＋低銷售量。每個情境可包含多個變動儲存格的值。
  • 變動儲存格 (Changing Cells)：情境中會改變數值的輸入儲存格，最多可設定 32 個。
  • 情境摘要 (Scenario Summary)：將所有情境的輸入值與關鍵輸出值並排成報表，方便決策者一次比較所有可能結果。

實務範例：三種市場情境的年度利潤比較

沿用損益分析模型，但現在有三種市場情境假設，每種情境下售價與銷售數量不同：

情境名稱 售價（B1） 銷售數量（B4） 預期利潤
🟢 樂觀情境 $90 5,000 單位 $210,000
🟡 普通情境 $80 3,500 單位 $2,500
🔴 悲觀情境 $70 2,000 單位 −$70,000

操作步驟：建立情境

  1. 點擊「資料」→「模擬分析」→「情境管理器 (Scenario Manager)」。
  2. 點擊「新增 (Add)」按鈕，建立第一個情境。
  3. 在「新增情境」對話框：
    • 情境名稱：輸入 樂觀情境
    • 變動儲存格：按住 Ctrl 同時點選 B1 和 B4（售價與數量）
    點擊「確定」。
  4. 在「情境值」對話框中輸入樂觀情境的數值：B1 = 90，B4 = 5000。點擊「新增」繼續建立下一個情境。
  5. 重複步驟 3–4，分別建立「普通情境」（B1=80, B4=3500）與「悲觀情境」（B1=70, B4=2000）。
  6. 全部情境建立完成後，點擊「關閉」。

操作步驟：切換情境 & 產生摘要報表

  1. 再次開啟情境管理器，在清單中選取任一情境，點擊「顯示 (Show)」，工作表數值會立即切換為該情境的假設值。
  2. 若要產生比較報表，點擊「摘要 (Summary)」按鈕。
  3. 在「情境摘要」對話框中：
    • 報表類型選「情境摘要 (Scenario Summary)」。
    • 結果儲存格：點選 B7（利潤），可按住 Ctrl 多選多個輸出儲存格。
    點擊「確定」。
  4. Excel 自動在新工作表中產生並排的比較報表，可直接印出或呈報使用。
💡 情境管理器 vs. 資料表：如何選擇？
  • 若要測試連續的數值範圍（如售價每次增加 $10），且需要一次性看到所有組合結果 → 使用資料表。
  • 若要儲存有意義的命名情境（如「樂觀」「悲觀」「基線」），並能隨時切換套用到工作表 → 使用情境管理器。
  • 情境管理器可改變最多 32 個輸入儲存格；資料表最多同時改變 2 個變數。

練習題 1：目標搜尋 — 生產線達標所需機器速度

任務描述：

某工廠的生產線每天運作 16 小時，目前機器速度為 80 件/小時， 但因設備閒置與換線，實際效率僅有 85%。 工廠接到訂單，必須在一天內生產至少 1,500 件成品。 請使用目標搜尋，找出機器速度需要達到多少才能完成訂單。

儲存格 項目 數值 / 公式
B1每日生產時數（小時）16
B2機器速度（件/小時）80（← 目標搜尋將調整此值）
B3實際效率85%
B4日產量（件）=B1*B2*B3
💡 專有名詞解釋
  • 機器速度 (Machine Speed / Cycle Rate)：生產線每單位時間可完成的產品數量，是 IE（工業工程）中衡量設備能力的關鍵指標。
  • 實際效率 (Overall Equipment Effectiveness, OEE)：考量設備停機、換線損失後，設備實際有效運轉時間與理論最大時間的比率。
  • 日產量：每日實際生產的成品數量，公式為：生產時數 × 機器速度 × 實際效率。

操作步驟：

  1. 依上表在工作表中建立資料，B4 輸入公式 =B1*B2*B3。
  2. 點擊「資料 → 模擬分析 → 目標搜尋」。
  3. 設定目標儲存格：B4，目標值：1500，變更儲存格：B2。
  4. 點擊「確定」，記錄 Excel 求得的機器速度結果。
  5. 驗算：將求得的速度代入公式，確認日產量是否恰好等於 1,500 件。
  6. 進階思考：若機器速度提升有上限（最高 130 件/小時），而實際效率最多可改善至 95%，請再次使用目標搜尋，分別求出在速度上限下，效率需達多少才能完成 1,500 件。

練習題 2：雙變數資料表 — 定價與銷量的利潤敏感性分析

任務描述：

某新創公司推出一款 APP 訂閱服務，財務模型如下。行銷部門估計不同定價與不同訂閱用戶數會有不同利潤， 請建立一個雙變數資料表，同時分析月訂閱費（$5–$25）與月活躍用戶數（5,000–25,000） 對每月利潤的影響，並找出利潤為正的最小定價與用戶數組合。

儲存格 項目 數值 / 公式
B1月訂閱費（元/用戶）$15
B2月活躍用戶數（人）12,000
B3每用戶月均服務成本（元）$4
B4每月固定營運成本（元）$60,000
B5月收入（元）=B1*B2
B6月變動成本（元）=B3*B2
B7月利潤（元）=B5-B6-B4
💡 專有名詞解釋
  • 月活躍用戶 (Monthly Active Users, MAU)：在一個月內至少使用一次該服務的用戶數量，是訂閱制 SaaS 公司衡量業務規模的核心指標。
  • 敏感性分析 (Sensitivity Analysis)：分析當輸入變數改變時，輸出結果變化幅度的方法。雙變數資料表是在 Excel 中進行敏感性分析最有效率的工具。
  • 損益平衡 (Break-Even)：在資料表矩陣中，利潤 = 0 的格子所在的組合即為損益平衡點；正值的格子代表獲利情境。

操作步驟：

  1. 依上表建立財務模型，確認 B7 的公式正確。
  2. 在新的區域（如 D1）輸入 =B7（公式儲存格，放在矩陣左上角）。
  3. 在 D1 右側（E1:I1）依序輸入列方向變數值（月訂閱費）：5、10、15、20、25。
  4. 在 D1 下方（D2:D6）依序輸入欄方向變數值（用戶數）：5000、10000、15000、20000、25000。
  5. 框選整個矩陣範圍 D1:I6，點擊「資料 → 模擬分析 → 資料表」。
  6. 列輸入儲存格：B1（月訂閱費）；欄輸入儲存格：B2（用戶數）。點擊「確定」。
  7. 分析結果：找出矩陣中所有利潤 ≥ 0 的組合，並使用條件式格式設定 (Conditional Formatting) 將正值格子標記為綠色、負值標記為紅色，讓損益平衡線一目了然。
  8. 進階挑戰：若公司要求每月利潤至少達 $50,000，在月訂閱費 $15 固定的情況下，最少需要多少活躍用戶？請利用資料表結果或目標搜尋工具驗證。

練習題 3：情境管理器 — 三種生產策略的年度成本比較

任務描述：

某製造廠正在評估三種生產策略（正常生產、加班生產、外包生產）， 每種策略的勞工成本、材料成本與固定費用不同。 請使用情境管理器建立三種情境，並產生情境摘要報表，比較各策略下的年度總成本與毛利。

儲存格 項目 基準值 說明
B1年產量（件）50,000各情境均固定
B2單位售價（元）$200各情境均固定
B3單位勞工成本（元）$45情境 1 變動儲存格
B4單位材料成本（元）$60情境 2 變動儲存格
B5年度固定成本（元）$1,500,000情境 3 變動儲存格
B6年度總收入（元）=B1*B2公式

  • 🟡 策略 B（加班生產）： 勞工成本 $65（加班費較高）/ 材料成本 $60 / 固定成本 $1,200,000（無需增設設備）
  • 🔴 策略 C（外包生產）： 勞工成本 $35（外包較便宜）/ 材料成本 $75（外包商採購成本較高）/ 固定成本 $800,000（廠內固定設備減少）
💡 專有名詞解釋
  • 毛利 (Gross Profit)：總收入扣除所有生產相關成本（含固定成本與變動成本）後的剩餘利潤，是衡量生產策略獲利能力的指標。
  • 加班生產 (Overtime Production)：利用現有設備與人員在正常工時以外延長工作，可提升產量但勞工成本大幅增加。
  • 外包生產 (Outsourcing)：將部分或全部生產委外給其他廠商，可降低廠內固定成本，但可能增加單位材料成本並降低品質管控能力。

操作步驟：

  1. 依上表建立財務模型，確認 B6、B7、B8 的公式均正確。
  2. 點擊「資料 → 模擬分析 → 情境管理器」，點擊「新增」。
  3. 建立「策略A_正常生產」情境：變動儲存格選 B3,B4,B5；數值分別為 45、60、1500000。
  4. 繼續新增「策略B_加班生產」情境：B3 = 65，B4 = 60，B5 = 1200000。
  5. 新增「策略C_外包生產」情境：B3 = 35，B4 = 75，B5 = 800000。
  6. 全部情境建立後，點擊「摘要 (Summary)」，結果儲存格選 B8（年度毛利）。點擊「確定」。
  7. 在 Excel 自動產生的情境摘要工作表中，比較三種策略的年度毛利，判斷哪種策略最有利可圖。
  8. 進階挑戰：若公司對外包商的單位材料成本有談判空間，從 $75 降低到 $65，請更新「策略C」的情境值，重新產生摘要報表，觀察結論是否改變。

🎓 學習重點回顧

  • 開啟位置：三種工具都在「資料 → 模擬分析 (What-If Analysis)」下拉選單中。
  • 目標搜尋：已知輸出目標，反求輸入值；每次僅能調整 1 個變數；結果直接寫入儲存格。
  • 資料表：一次計算 1–2 個變數在所有指定值下的輸出，自動填入結果矩陣；輸出為陣列公式，不可單獨修改。
  • 情境管理器：儲存最多 32 個輸入儲存格的多組命名假設；可切換顯示或一鍵產生並排比較的摘要報表。
  • 損益平衡點：固定成本 ÷（單位售價 − 單位變動成本），可用目標搜尋快速驗算。
  • 工具選擇原則：逆向求解 → 目標搜尋；連續值域矩陣 → 資料表；命名情境比較 → 情境管理器。
