📊 巨集 (Macros) 與報表自動化

錄製巨集與基礎 VBA 編輯，實現一鍵自動化報表生成

1什麼是巨集 (Macro)?

巨集是 Excel 中一種強大的自動化工具，能夠將重複性的操作步驟錄製下來，並在需要時一鍵執行，大幅提升工作效率。

💡 專有名詞解釋：
  • 巨集 (Macro)： 一組預先錄製或編寫的操作指令，可以自動執行重複性任務，如資料整理、格式設定、報表生成等。
  • VBA (Visual Basic for Applications)： Excel 內建的程式語言，用於編寫和編輯巨集。即使不懂程式設計，也可以透過「錄製巨集」功能自動生成 VBA 程式碼。
  • 巨集錄製器： Excel 提供的工具，可以記錄您的操作步驟並自動轉換為 VBA 程式碼。

📌 巨集的應用場景

  • 報表自動化：每月固定格式的銷售報表、財務報表生成
  • 資料整理：批次刪除空白列、統一格式、資料驗證
  • 樞紐分析表更新：自動更新多個樞紐分析表並刷新資料
  • 匯出作業：將特定工作表另存為 PDF 或其他格式
  • 重複性操作：任何需要多次執行的固定流程
⚠️ 注意事項：
  • 巨集會記錄您的每一個操作步驟，包括錯誤操作，因此錄製前建議先規劃好流程。
  • 巨集檔案需要儲存為 .xlsm 格式（啟用巨集的 Excel 活頁簿）。
  • 第一次執行巨集時，Excel 可能會顯示安全性警告，需要啟用巨集才能執行。

2如何錄製巨集

Excel 提供了簡單易用的「錄製巨集」功能，讓您無需撰寫程式碼即可創建自動化流程。

🎬 錄製巨集的步驟

步驟 1：啟用開發人員索引標籤
  1. 點選 檔案 → 選項 → 自訂功能區
  2. 在右側勾選「開發人員」選項
  3. 按下「確定」，功能區即會顯示「開發人員」索引標籤
步驟 2：開始錄製巨集
  1. 點選 開發人員 → 錄製巨集
  2. 設定巨集名稱（例如：生成月報表）
  3. 選擇「儲存位置」：
    • 此活頁簿：僅在目前檔案中可用
    • 新活頁簿：每次開新檔案都要重新錄製
    • 個人巨集活頁簿：所有 Excel 檔案都可使用（建議選此項）
  4. （選用）設定快速鍵，例如 Ctrl + Shift + M
  5. 按下「確定」開始錄製
步驟 3：執行要自動化的操作

錄製開始後，Excel 會記錄您的每一個動作。例如：

  • 調整欄寬、列高
  • 套用格式（加粗、顏色、框線）
  • 插入公式或函數
  • 建立或更新樞紐分析表
  • 另存新檔為 PDF
步驟 4：停止錄製
  1. 完成所有操作後，點選 開發人員 → 停止錄製
  2. 巨集已成功建立，可在「開發人員 → 巨集」中找到
💡 錄製技巧：
  • 錄製前先在空白檔案練習一次流程，確保步驟正確無誤。
  • 盡量使用鍵盤快速鍵（如 Ctrl + Home 回到 A1 儲存格），而非滑鼠點選，巨集執行會更穩定。
  • 避免在錄製過程中切換工作表或檔案，除非這是流程的一部分。

3執行與管理巨集

▶️ 如何執行巨集

有多種方式可以執行已錄製的巨集：

方法 1：透過功能區

點選 開發人員 → 巨集，選擇巨集名稱後按「執行」。

方法 2：使用快速鍵

如果錄製時有設定快速鍵（例如 Ctrl + Shift + M），直接按快速鍵即可執行。

方法 3：按鈕觸發

在工作表中插入按鈕（開發人員 → 插入 → 按鈕），指定巨集後即可一鍵執行。

🗂️ 管理巨集

  • 檢視巨集清單：點選 開發人員 → 巨集，即可看到所有可用的巨集。
  • 編輯巨集：選擇巨集後點選「編輯」，進入 VBA 編輯器查看或修改程式碼。
  • 刪除巨集：選擇巨集後點選「刪除」即可移除不需要的巨集。
  • 重新命名巨集：需進入 VBA 編輯器，修改 Sub 巨集名稱() 中的名稱。
⚠️ 儲存格式注意：

含有巨集的 Excel 檔案必須儲存為 .xlsm 格式。如果儲存為一般的 .xlsx 格式，巨集將會遺失。

4基礎 VBA 編輯

錄製的巨集會自動轉換為 VBA 程式碼。了解基礎的 VBA 語法，可以幫助您更靈活地調整和優化巨集。

🖥️ 開啟 VBA 編輯器

  • 點選 開發人員 → Visual Basic
  • 或按快速鍵 Alt + F11

📝 VBA 程式碼結構範例

範例：簡單的格式設定巨集

Sub 設定標題格式() '選取 A1 儲存格 Range("A1").Select '設定字型為粗體、大小 16、顏色為藍色 With Selection.Font .Bold = True .Size = 16 .Color = RGB(0, 0, 255) End With '設定儲存格背景顏色為淺灰色 Selection.Interior.Color = RGB(220, 220, 220) '提示訊息 MsgBox "標題格式設定完成！" End Sub
💡 VBA 基礎語法說明：
  • Sub ... End Sub： 定義一個巨集（副程式）的開始與結束。
  • Range("A1")： 指定儲存格範圍，例如 Range("A1:D10") 表示 A1 到 D10 的範圍。
  • Selection： 代表目前選取的儲存格或範圍。
  • With ... End With： 簡化多個屬性設定，避免重複撰寫物件名稱。
  • MsgBox： 顯示訊息對話框。
  • ' (單引號)： 註解符號，該行內容不會被執行，用於說明程式碼功能。

🔧 常用的 VBA 修改技巧

技巧 1：修改儲存格範圍

錄製的巨集可能會固定選取特定儲存格，您可以修改為動態範圍：

'原始錄製（固定選取 A1:A10） Range("A1:A10").Select '修改為動態選取（選取 A 欄所有有資料的儲存格） Range("A1", Range("A1").End(xlDown)).Select

技巧 2：新增變數提升彈性

使用變數可讓巨集更靈活，避免寫死特定數值：

Sub 計算總和() Dim 總和 As Double 總和 = WorksheetFunction.Sum(Range("B2:B100")) Range("B101").Value = 總和 MsgBox "總和為：" & 總和 End Sub

技巧 3：加入條件判斷

讓巨集根據不同狀況執行不同動作：

Sub 檢查銷售額() If Range("D2").Value > 100000 Then MsgBox "恭喜！本月銷售達標！" Else MsgBox "尚未達標，請繼續努力！" End If End Sub

5實戰案例：一鍵生成月報表

以下將示範如何錄製一個完整的月報表自動化巨集，包含更新樞紐分析表、格式調整及匯出 PDF。

📋 月報表自動化流程

步驟 1：準備資料
  • 假設您有一個包含銷售資料的工作表「原始資料」
  • 另有一個工作表「月報表」，其中包含樞紐分析表
步驟 2：開始錄製巨集
  1. 點選 開發人員 → 錄製巨集
  2. 巨集名稱：生成月報表
  3. 快速鍵：Ctrl + Shift + R
  4. 儲存位置：此活頁簿
步驟 3：執行自動化操作
  1. 更新樞紐分析表：
    • 切換到「月報表」工作表
    • 點選樞紐分析表內任一儲存格
    • 右鍵選擇「重新整理」
  2. 調整報表格式：
    • 選取標題列（例如 A1:F1），設定粗體、背景色、字型大小
    • 調整欄寬為自動調整（雙擊欄位邊界）
    • 設定數值格式（例如千分位符號、小數點）
  3. 新增報表日期：
    • 在特定儲存格（例如 A1）輸入 =TODAY()
    • 設定日期格式為「yyyy/mm/dd」
  4. 匯出為 PDF：
    • 點選 檔案 → 另存新檔
    • 選擇儲存類型為「PDF」
    • 檔案名稱設定為「月報表_年月.pdf」（例如：月報表_2024年12月.pdf）
    • 按下「儲存」
步驟 4：停止錄製並測試
  1. 點選 開發人員 → 停止錄製
  2. 測試巨集：按 Ctrl + Shift + R 或透過「巨集」清單執行
  3. 確認報表是否正確更新並成功匯出 PDF

自動生成的 VBA 程式碼範例（部分）

Sub 生成月報表() '切換到月報表工作表 Sheets("月報表").Select '重新整理樞紐分析表 ActiveSheet.PivotTables("樞紐分析表1").PivotCache.Refresh '設定標題格式 Range("A1:F1").Select With Selection.Font .Bold = True .Size = 14 .Color = RGB(255, 255, 255) End With Selection.Interior.Color = RGB(68, 114, 196) '插入報表日期 Range("H1").Value = "報表日期：" Range("I1").Value = Date '匯出為 PDF ActiveSheet.ExportAsFixedFormat Type:=xlTypePDF, _ Filename:="C:\Reports\月報表_" & Format(Date, "yyyy年mm月") & ".pdf" MsgBox "月報表已成功生成並儲存為 PDF！" End Sub
💡 進階優化建議：
  • 將 PDF 儲存路徑改為變數，方便日後修改。
  • 加入錯誤處理機制（On Error Resume Next），避免執行中斷。
  • 使用迴圈批次處理多個樞紐分析表或工作表。

📝 練習題

練習題 1：錄製資料清理巨集

任務描述：

您每週都會收到一份格式混亂的銷售資料表，需要進行以下清理步驟：

  • 刪除所有空白列
  • 將標題列（第一列）設定為粗體、藍色背景、白色字體
  • 將所有金額欄位設定為千分位格式，並保留兩位小數
  • 自動調整所有欄寬至最適大小
  • 在 A1 儲存格上方插入一列，輸入「資料清理日期：」和今天日期

練習要求：

  1. 請錄製一個名為「資料清理」的巨集，完成以上所有步驟。
  2. 設定快速鍵為 Ctrl + Shift + C
  3. 在工作表中插入一個按鈕，並指定執行此巨集。
  4. 測試巨集是否能正確執行所有清理步驟。
💡 提示：
  • 刪除空白列：選取整個工作表 → 常用 → 尋找與選取 → 到特殊目標 → 空白 → 刪除列
  • 調整欄寬：選取所有欄 → 常用 → 格式 → 自動調整欄寬
  • 插入列：右鍵點選列號 → 插入

練習題 2：建立樞紐分析表更新巨集

任務描述：

您的 Excel 檔案中有三個工作表：「原始資料」、「銷售分析」、「產品分析」。後兩個工作表各包含一個樞紐分析表，每當原始資料更新後，需要手動重新整理兩個樞紐分析表。請建立一個巨集來自動化此流程。

練習要求：

  1. 錄製一個名為「更新所有樞紐表」的巨集，執行以下操作：
    • 切換到「銷售分析」工作表，重新整理樞紐分析表
    • 切換到「產品分析」工作表，重新整理樞紐分析表
    • 顯示訊息「所有樞紐分析表已更新完成！」
    • 最後切換回「原始資料」工作表
  2. 設定快速鍵為 Ctrl + Shift + U
  3. 執行巨集並確認兩個樞紐分析表都已正確更新。
💡 進階挑戰：

開啟 VBA 編輯器（Alt + F11），查看自動生成的程式碼。嘗試修改程式碼，使其能夠自動偵測並更新活頁簿中的「所有」樞紐分析表，而不是只更新特定的兩個。

提示：可以使用迴圈遍歷所有工作表和樞紐分析表。

練習題 3：一鍵生成與寄送週報表

任務描述：

公司要求每週五下班前提交週報表 PDF 給主管。報表需包含本週銷售統計、排名前五的產品、以及與上週的比較分析。目前這些步驟都需要手動完成，非常耗時。請建立一個完整的自動化流程。

練習要求：

  1. 錄製一個名為「生成週報表」的巨集，執行以下步驟：
    • 更新「週報表」工作表中的樞紐分析表
    • 在報表標題處（例如 A1）自動填入當週日期範圍（例如：2024/12/09 - 2024/12/15）
    • 調整表格格式：
      • 標題列：粗體、18pt、深藍色背景、白色字體
      • 數值欄位：千分位、兩位小數
      • 自動調整欄寬
    • 將「週報表」工作表匯出為 PDF，檔案名稱為「週報表_YYYYMMDD.pdf」（例如：週報表_20241215.pdf）
    • 顯示訊息「週報表已生成！檔案儲存於：[檔案路徑]」
  2. 設定快速鍵為 Ctrl + Shift + W
  3. 在工作表中建立一個美觀的按鈕「🚀 生成週報表」，並指定此巨集。
⚠️ 進階挑戰（選做）：

在 VBA 編輯器中加入以下功能：

  • 檢查今天是否為週五，如果不是則顯示提示訊息「今天不是週五，確定要生成週報嗎？」
  • 自動開啟 Outlook 並建立一封新郵件，收件人為主管 Email，主旨為「週報表_YYYYMMDD」，並自動附加剛才生成的 PDF 檔案。

提示：需要學習使用 VBA 的 Outlook 物件模型和檔案操作函數。

💡 提示：
  • 使用 =TODAY() 函數取得今天日期
  • 使用 =TEXT(TODAY(), "YYYYMMDD") 格式化日期為檔案名稱
  • 匯出 PDF 時，在「另存新檔」對話框中選擇存檔類型為「PDF」
  • VBA 中使用 Format(Date, "YYYYMMDD") 格式化日期
